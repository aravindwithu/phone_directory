Used C language

Phone directory system which does the following:

-> Insert the provided contact data.

-> Sort the contact using quick sort algorithm.

-> Search the contact using binary search algorithm to retrieve information about given name. quick sort must be done before binary search is done.

-> Identify the index of the given name using binary search and then using the given index the delete operation is done. Once the identified index is deleted then a quick sort is performed to retain the sort order.
